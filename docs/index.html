Minecraftclone’s software design documentation by kruslin2

These following modules together with all other modules which are tested/documented by other Team members are used/imported by other bigger modules to form the minecraft program code which will generate terrain and skies with features such as moving clouds.
For a reference, the actuall minecraft program code developed using python3 by team member kruslin2, which uses these code modules, can be downloaded from this repository on branch minecraftclone.
